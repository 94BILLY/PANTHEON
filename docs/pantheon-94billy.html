Pantheon is Path 1 on real silicon: the EE streams DMA and VIF work; VU1 microcode transforms vertices, packs GIF primitives, and XGKICKs the Graphics Synthesizer—no thick software renderer in between.

This repository is a public technical record and a reproducible whitebox (boot, skydome, floor, camera)—not a starter kit and not permission to ship derivatives.

Build floor.elf with PS2SDK, run it in PCSX2 or on hardware, and read the contract in pantheon_path1_contract.h next to the microcode in shader.vsm.

shader.vsm — entry (VU1)
.vu
.global PantheonShader
PantheonShader:
; Path 1 shared contract:
;   giftag @ 240, output @ 241, xgkick @ 240, input @ 8
nop lq VF06, 240(VI00) ; GIF tag
nop lq VF05, 2(VI00)   ; GS center (2048,2048); Z bias
nop lq VF07, 3(VI00)   ; XY scale, Z scale, nearZ, maxXY
nop lq VF01, 4(VI00)   ; MVP row 0
nop lq VF02, 5(VI00)   ; MVP row 1
nop lq VF03, 6(VI00)   ; MVP row 2
nop lq VF04, 7(VI00)   ; MVP row 3
nop sq VF06, 240(VI00)
nop xitop VI01
VertexLoop:
nop ibeq VI01, VI00, EndLoop
; … transform, pack, xgkick …
