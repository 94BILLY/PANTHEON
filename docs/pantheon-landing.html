PANTHEON

PlayStation 2 · Path 1 · VU1

“The Emotion Engine as conductor. VU1 as geometry worker. DMA/VIF as law.”

v1.0.0-beta — Path 1 baseline

Pantheon is Path 1 on real silicon vocabulary: EE schedules DMA/VIF1; VU1 microcode transforms, packs GIF, XGKICK to GS. Not a tutorial repo. If that sentence is opaque, the manuals come first—then this tree reads as what it is.

Public showcase of the work—not a starter kit, not permission to ship derivatives. No LICENSE; rights reserved. In-repo tools: hrc2ps2.py (Softimage .hrc), obj2ps2.py (Wavefront .obj)—integration with the Path 1 contract is on you.

This baseline is a technical record and reproducible whitebox—not a product pitch, not “make your own PS2 game,” and not permission to treat the tree as a template for unrelated shipping work.

1. What’s in this baseline

  • Path 1 pipeline: EE → VIF1 → VU1 → GIF → GS.
  • Mesh crunchers in-tree: hrc2ps2.py, obj2ps2.py.
  • Skydome + timecycle; hybrid floor vs strict Path1 floor (see GitHub BETA_RELEASE.md).
  • Boot overlay + 60 FPS-class whitebox target; contract in headers.
