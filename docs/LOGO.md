# Pantheon mark (reference)

File: **`pantheon-logo-mark-timeless.png`**

- Use as **reference art** for favicons, headers, and print.
- For **production** (trademark filing, large-format print, crisp scaling), **redraw in vector** (SVG or Illustrator) from this geometry: three columns + pediment, single flat color.
- **Wordmark** (optional): set “PANTHEON” in a neutral grotesk (e.g. similar to Helvetica / Neue Haas) with wide tracking; keep the icon and type **separate** so the mark works alone at 16×16.
